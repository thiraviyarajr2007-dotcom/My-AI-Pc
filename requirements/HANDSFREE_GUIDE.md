## 🎤 VOICE COMMANDS

### Application Control:
```
"open chrome"          → Opens Chrome browser
"open notepad"         → Opens Notepad
"open calculator"      → Opens Calculator
"open file explorer"   → Opens File Explorer
"open vscode"          → Opens VS Code
"open spotify"         → Opens Spotify
```

### File Operations:
```
"find my documents"           → Searches for documents
"search for python files"     → Finds *.py files
"find files containing TODO"  → Content search
"show recent files"           → Lists recent files
```

### Mouse Control:
```
"click"                       → Click mouse
"double click"                → Double click
"right click"                 → Right click
"move mouse to 500 300"       → Move to position
"move mouse to center"        → Move to screen center
```

### Keyboard Control:
```
"type hello world"            → Types text
"press enter"                 → Presses Enter key
"press ctrl s"                → Saves (Ctrl+S)
"press alt tab"               → Switches windows
```

### Screen Capture:
```
"take screenshot"             → Captures screen
"take a screenshot"           → Captures screen
"screenshot"                  → Captures screen
```

### System Commands:
```
"lock screen"                 → Locks computer
"volume up"                   → Increases volume
"volume down"                 → Decreases volume
"mute"                        → Toggles mute
```

### Scrolling:
```
"scroll up"                   → Scrolls up
"scroll down"                 → Scrolls down
"scroll up 5"                 → Scrolls up 5 units
```

### Gesture Controls:
```
0 fingers (Fist)       → No action
1 finger (Point)       → Open Chrome
2 fingers (Peace)      → Take screenshot
3 fingers              → Open Notepad
4 fingers              → Open Calculator
5 fingers (Open palm)  → Open File Explorer
```

### How to Use Gestures:
1. Make sure your webcam is connected
2. Hold gesture for 2-3 seconds
3. Action executes automatically

## 🔧 TROUBLESHOOTING

### Voice Not Working:
```
1. Check microphone is connected
2. Allow microphone in browser
3. Speak clearly and slowly
4. Check Whisper model loaded
5. Try text input first
```

### Commands Not Executing:
```
1. Check AI Brain status (should be Active)
2. Verify Ollama is running: ollama serve
3. Check console for errors
4. Restart the system
```

### Gestures Not Working:
```
1. Check webcam is connected
2. Ensure good lighting
3. Hold gesture for 2-3 seconds
4. Check MediaPipe installed
```

## 🎓 TIPS FOR BEST RESULTS

### Voice Commands:
1. **Speak clearly** - Enunciate words
2. **Speak at normal pace** - Not too fast or slow
3. **Use simple commands** - "open chrome" not "could you please open chrome"
4. **Wait for response** - Let system process before next command
5. **Use exact app names** - "chrome" not "google chrome"

### Gesture Control:
1. **Good lighting** - Ensure face/hands are well-lit
2. **Hold steady** - Keep gesture for 2-3 seconds
3. **Clear background** - Avoid cluttered backgrounds
4. **One hand** - Use one hand at a time

### General Tips:
1. **Start simple** - Try basic commands first
2. **Learn patterns** - System learns your usage
3. **Use workflows** - Automate repetitive tasks
4. **Check status** - Monitor system status in sidebar
5. **Practice** - Gets better with use
