  → Check Ollama running: ollama serve
  → Check port 5000 free
  → Restart server

  Voice not working:
  → Allow microphone in browser
  → Check Whisper model loaded
  → Try text input first

  Slow performance:
  → Close unused apps
  → Use smaller Whisper model
  → Clear old data
