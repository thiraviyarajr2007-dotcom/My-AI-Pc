## 🔧 Troubleshooting

### Common Issues

**"Module not found"**
```bash
pip install -r requirements.txt
```

**"Ollama not connecting"**
```bash
ollama serve
ollama pull llama3
```

**"Microphone not working"**
- Check system permissions
- Test with: `python voice_ai.py`

**"PyAudio error"**
```bash
pip install pipwin
pipwin install pyaudio
```
