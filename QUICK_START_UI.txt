┌────────────────────────────────────────────────────────────────┐
│  🔧 TROUBLESHOOTING                                            │
└────────────────────────────────────────────────────────────────┘

  UI not loading?
  → pip install flask flask-socketio
  → python christa_ui.py

  Voice not working?
  → Allow microphone in browser
  → Check Whisper model loaded
  → Try text input first

  Port in use?
  → netstat -ano | findstr :5000
  → taskkill /PID <PID> /F
